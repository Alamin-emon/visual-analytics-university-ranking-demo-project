# 📊 visual-analytics-university-ranking-demo-project

Demo project for the **Visual Analytics course** in the Master’s program in Data Science and Business Informatics at the University of Pisa (Unipi).

---

## 🚀 Project Overview

Choosing a suitable university is a major decision for students, especially when planning to study abroad.  
The process becomes easier when students have access to clear, interactive tools that support informed decision-making.

This project aims to solve this problem by providing a **digital and interactive visualization tool** to explore and compare universities worldwide.

The system allows users to analyze the **top 400 universities globally**, using historical ranking data.

---

## 🎯 Objective

The main objective of this project is to help students explore and compare universities using data-driven visual analytics.

The dataset includes the **Times Higher Education (THE) World University Rankings**, covering the top 400 universities from **2011 to 2016**.

---

## 📊 Dataset Used

- **Times Higher Education (THE) World University Ranking**
  - Top 400 universities per year (2011–2016)

- **NUMBEO Dataset**
  - Cost of living data for cities and countries

## ▶️ How to Run the Project

To run the application locally, follow these steps:

 Download the Repository
```bash

Navigate to the Project Folder
cd Visual-Analytics-project

Start a Local Server

Since this is a web-based project, it must be run using a local server.

Option A: Python Server
python -m http.server 8000

npx serve .
